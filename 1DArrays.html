Single Arrays


Single Arrays are the most basic data structure you can find. These are immutable, meaning that once we initialize a length for them, the size of the array cannot be altered. The number of data we put In cannot exceed the length of the Array. Arrays are the only data type that can hold both primitives (String, int, boolean) and objects. We can initialize arrays in two ways.

Intialization

There is no importation for Arrays


int [] exampleArray = {1,2,3,4,5};

Or

int [] exampleArray = new int{1,2,3,4,5};


The brackets tell the computer that we are going to make an array, and the 'int' right before the brackets tell the computer what type of data we are going to put in the array. The name is given right after the brackets. The numbers inside the curly brackets are the ints that the array starts out with. We can change these values, which I will show later , but we cannot add anything else to the array.

We can also intialize an Array like this


int [] exampleArray = new int[3];


The noticeable difference here is the 'new int[3]'. The “new int []” portion tells the computer to create a new int Array, while the 3 tells the computer to make this Array have 3 spots. When we initialize an Array like this, all of the 3 spots are filled with 0 as place holders. It looks like this.


0
0
0



If we wanted to fill these with actual values, say from 1 to 3,this is what we would do .


exampleArray[0] = 1;

exampleArray[1] = 2;

exampleArray[2] = 3;


Now our example array would look like this


1
2
3




We could have easily filled this up with a for or while loop as well.

As you noticed, the first element in this Array is referenced to as zero instead of one. This rule applies for any data set that uses numbers to iterate. So if you want to call the first element in an Array, use zero instead of one. We can also replace values in our Array the same way that we filled it up. Say we wanted to turn the 1 in the Array into a 4. All we would have to do is this.


exampleArray[0] = 4;


Now our Array would look like this


4
2
3

Iteration

Iteration through Arrays is fairly simple. One way to do it is with a traditional for loop, as seen below


for(int i= 0;i < exampleArray.length; i++)

{

//code

}


We start the iterator at zero because our first element in the array is indexed at zero, and we make sure that it is less than instead of less than or equal to in the second parameter for the same reason. If it was less than or equal to, then i would be a number that exceeds the carrying capacity of the Array. We do 'i++'in the next parameter in order to iterate through each element one by one.

You can also iterate with a for- each loop, as shown below


for(int a : exampleArray)

{

//code

}


Here, the code looks in every int in exampleArray and references them as multiple integers called 'a'. You can compare values easier with this method.
