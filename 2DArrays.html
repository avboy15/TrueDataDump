Two Dimensional Arrays


In its simplest form, a 2D Array is an Array that holds Arrays. It sounds complex, but it is pretty useful and versatile. 2D Arrays share all the characteristics and methods of single dimension arrays.

Intialization

There is no importation for Arrays.

The initialization of a 2D Array looks like this


int[][] exampleArray = {{1,2},{3,4}};

and

Int [][] exampleArray = new int[][]{{1,2},{3,4}};

Or like this

Int[][] exampleArray = new int[2][2]; (This creates an empty array with the given dimensions)


To make this simple, think of 2D arrays as a table of values, with the first [] representing rows and the second [] representing columns. The exampleArray with values would look like this


1
2
3
4






So if we wanted to call the element number 3, whether to change it or print it out, we would do this


exampleArray [1][0];


By adding another [] when we are creating the array, we can make a 3D Array and so on. However, as we add more dimensions to Arrays, they become more and more complex to work with.

Iteration

This is similar to single dimensional arrays, but we will have to embed a for loop within a for loop, like this.


for(int i = 0; i < exampleArray.length; i++)

{

for(int y = 0; y < exampleArray[i].length; y++)

{

//code

}

}


The outermost for loop iterates through each separate Array in the 2D Array, while the inner for loop looks at the individual elements in the current Array that the outer loop is looking at. We put y < exampleArray[i].length as a parameter in the second for loop because it tells the computer to look at every element at whatever Array the outer loop is currently on. However, we could have also put y < exampleArray[0].length, because Arrays have to be rectangular, so the length of one of the arrays is the length of all of them.

We can also use a for-each loop, like this


for(int[] y : exampleArray)

{

for(int a : y)

{

//code

}

}


The outer loop's parameter looks at every individual Array 'y' in the exampleArray, and we already know what the inner loop does.

We used int Arrays for all the example, but all of this could have been done with Arrays of any primitive or User made object. All we would have to do is replace the 'int' calls with the object's name and made the input that object type, as shown with string below


String [] stringArray = new String [2];

String [] stringArray = {'hello','there'};
