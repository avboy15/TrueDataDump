How this website works


This page will give you an idea of how each data structure in this website is explained. Each Data Structure has it’s own page, and each page has 3 sections. Before any of the sections, there is a short introduction that gives a brief explanation of how that Data Structure holds data. The first section,”Initialization”, gives examples on how to instantiate that Data Structure and explains each separate part of the Initialization. The section after this, “Useful methods”, is a list of the most helpful methods of that Data Structure, along with a link to the API of that Data Structure, which contains all the methods for that Class. The last section, “iteration”, gives a short portion of code that shows you how to iterate through that Data Structure. However, in some cases, this code shows you a method of the class of that Data Structure. The method iterates through the Data Structure. In summary, the iteration section gives you a better idea of the inner workings of the Data Structure. There will also be some words in red and green that describe code, like this and this. If the word is green, then you can scroll your cursor over the red word and a the part of the code that it is describing will also be highlighted in red as long as your cursor is over the original red word. If the word is green, then you can press the word and the code that it is describing will be highlighted in red. If you click the word again, then the word will unhighlight.
