What are Data Structures?

Data structures are the most important thing that you will find in the world of Computer Science, no matter your field. They hold all sorts of data, from primitive ints, strings, booleans, and chars, to user made objects. Although this sounds simples, there are numerous data structures in the world of CS, each unique but all interrelated. This website will teach you about some of these data structures. To start off, check out'single array' and then work your way from there.

How this website works

However, in some cases, this code shows you a method of the class of that Data Structure. The method iterates through the Data Structure. If the word is green, then you can scroll your cursor over the red word and a the part of the code that it is describing will also be highlighted in red as long as your cursor is over the original red word. If the word is green, then you can press the word and the code that it is describing will be highlighted in red. If you click the word again, then the word will unhighlight.
